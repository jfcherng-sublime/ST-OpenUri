## [4.0.0] - 2019-07-30

### Changed
- Setting `on_hover` is now changed to `show_open_button`.
  `"on_hover": true` is the same with `"show_open_button": "hover"`.
  `"on_hover": false` is the same with `"show_open_button": "always"`.
